### Cara Ganti Template

#### 1. Dari Halaman Template Presets

Login ke dashboard landingin dengan mengetikan url app.landingin.id lalu klik icon **Themes** pada menu navigasi di dashboard
Lalu klik button **Impor Halaman** seperti screenshot dibawah ini :

<img src="/uploads/template-presets.png" alt="Alt text" width="100%" />

Setelah klik button **Impor Halaman** akan diarahkan ke halaman baru untuk memilih opsi :

1. Buat halaman baru sekaligus pasang template.
2. Pasang Template ke halaman yang sudah ada.



#### A. Apabila memilih opsi pasang template ke halaman yang sudah ada

Maka anda akan diberikan 2 pertanyaan ingin pasang di domain apa dan ingin dipasang di halaman mana terkait template tersebut.

<img src="/uploads/pilih-1.png" alt="Alt text" width="100%" />

#### B. Apabila memilih opsi buat halaman baru sekaligus pasang template

Maka anda akan diberikan 1 pertanyaan ingin dipasang di domain apa terkait template tersebut.

<img src="/uploads/pilih-2.png" alt="Alt text" width="100%" />

Apabila sudah menentukan pilihan silahkan klik button **Pilih** maka proses impor akan berlangsung.
