### Memulai

Silahkan login ke dashboard landingin terlebih dahulu dengan mengetikan url https://app.landingin.id lalu pergi ke halaman
**Landing Page -> Pilih Domain -> Pilih Halaman -> Klik Button "Pergi ke Editor"**

### Auto Login Editor

Setelah mengklik button "Pergi ke Editor" maka secara otomatis akan redirect ke halaman editor landingin
dan anda dapat mengedit konten website melalui editor yang telah disediakan.
