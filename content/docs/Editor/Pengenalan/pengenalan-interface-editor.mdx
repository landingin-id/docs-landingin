### 1. Interface Komponent Section
Tersedia kumpulan komponent element siap pakai seperti button, image, paragraph, text, input dan masih banyak lagi. Contoh screebshot editor bisa anda lihat pada gambar dibawah ini :

### 2. Interface Blocks Section
Block adalah kumpulan dari beberapa komponen menjadi satu. Tersedia kumpulan block siap pakai seperti column, boxed, card, listed dan masih banyak lagi. Contoh screebshot editor bisa anda lihat pada gambar dibawah ini :

### 3. Interface Library Section
Section adalah kumpulan dari beberapa block menjadi satu. Tersedia kumpulan section template siap pakai seperti hero section, about section, list section, galeri section, header section, produk section dan masih banyak lagi. Contoh screebshot editor bisa anda lihat pada gambar dibawah ini :
<img src="/uploads/interface-section.png" alt="Alt text" width="100%" />

### 4. Interface Navbar Section
Interface editor yang berada di posisi paling atas, yang fiturnya sendiri terdiri dari :

1. **Panel Hide or Show** : Menampilkan atau menyembunyikan panel interface baik itu sidebar panel left, atau panel properties
2. **Undo Redo** : Fitur untuk membalikan perubahan atau memajukan perubahan editing landing page yang baru saja dilakukan.
3. **Preview** : Fitur mode preview landing page yang hanya menampilkan landing page saja tanpa adanya interface editor.
4. **Fullscreen** : Mode fullscreen editor, apabila anda ingin menggunakan editor dalam mode fullscreen.
5. **Mode Responsive** : Untuk melihat tampilan website dalam mode desktop, tablet atau handphone.
6. **Save** : Fitur menyimpan perubahan landing page yang telah dilakukan.
7. **Zoom in/out** : Perbesar / perkecil pratinjau tampilan landing page
8. **Revision** : Fitur kontrol versi untuk membalikan ke versi perubahan sebelumnya.

### 5. Interface Properties
Interface properties adalah panel terkait pengaturan element block, element section dan element komponent. untuk detail pengaturan dapat
anda lihat pada link [pengenalan properties](/docs/Editor/Pengenalan/mengenal-pengaturan-properties-element).
