### Apa itu komponent

Website bukan cuman tampilan. ini mesin konversi anda. optimalkan dengan komponen yang tepat!. Komponent sendiri adalah element terkecil pada suatu landing page seperti button, text, header, image dan lain lain yang
membentuk sebuah informasi untuk dijadikan konversi penjualan.

Web builder kami adalah kanvas digital Anda. Setiap komponen adalah alat yang memberi Anda kebebasan desain penuh, dari nol hingga masterpiece.

### Komponent tersedia

Komponent sendiri dikelompokan menjadi 6 kelompok yang diantaranya terdapat kelompok:

1. Base
2. Elements
3. Embeds
4. Widgets
5. Bootstraps
6. Plugins

### Komponent Base
Komponent yang tersedia pada kelompok base terdiri dari heading, image, html button, paragraph, form, input, textarea, selectbox, checkbox, radio button, link, list, table, audio.

### Komponent Elements
Komponent yang tersedia pada kelompok elements baru terdiri dari SVG Icon dan Code, nanti akan ada update
penambahan lagi terkait kelompok ini. 

### Komponent Embeds
Komponent yang tersedia pada kelompok elements baru terdapat komponent embeds, nanti akan ada update
penambahan lagi terkait kelompok ini.

### Komponent Widgets 
Komponent yang tersedia pada kelompok widgets terdiri dari Google Maps, Embed Video, Open Street Map.
### Komponent Widgets 
Komponent yang tersedia pada kelompok bootstrap terdiri dari element element yang tersedia pada bootstrap yang terdiri dari :
container, grid row, button, button group, alert, card, list group, badge.

### Komponent Plugins
Editor landingin memiliki fitur install plugins yang saat ini fitur tersebut belum tersedia. dan akan tersedia
dalam waktu dekat, default plugin yang tersedia adalah Contact form.
