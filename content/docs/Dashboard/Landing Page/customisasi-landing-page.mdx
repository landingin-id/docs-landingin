Setiap anda membuat domain baru pada panduan [buat domain baru](/docs/Dashboard/Landing%20Page/buat-domain-baru) maka secara otomatis
akan menggenerate landing page dengan default template yang sudah di tentukan oleh sistem.

### Cara Kustomisasi

Pada tahapan ini katakanlah anda baru saja membuat domain baru dan ingin melakukan penyesuaian tampilan lading page sesuai konten yang anda memiliki,
anda dapat pergi ke halaman landing page lalu klik button **"Pergi Ke Editor"** seperti screenshot dibawah ini :

<img src="/uploads/customisasi-landing-page-1.png" alt="Alt text" width="80%" />

Maka secara otomatis anda akan diarahkan ke halaman editor landingin seperti screenshot dibawah ini :

<img src="/uploads/customisasi-landing-page-2.png" alt="Alt text" width="80%" />

Panduan terkait penggunaan editor bisa dilihat pada link berikut [panduan editor](/docs/Editor), pada
editor <br/> anda dapat melakukan kustomisasi tampilan secara menyeluruh beserta konten didalamnya.

### Cara kustomisasi per halaman

Tiap halaman yang sudah anda buat dapat di kustomisasi baik secara tampilan maupun isi konten'nya melalui editor dengan cara
pada halaman landing page pilih halaman yang ingin di kustomisasi lalu klik dropdown <br/>  **"Actions"** lalu pilih **kustomisasi** :

<img src="/uploads/customisasi-page-3.png" alt="Alt text" width="80%" />

Maka setelahnya anda akan secara otomatis di arahkan ke halaman editor terkait halaman yang ingin anda kustomisasi.
