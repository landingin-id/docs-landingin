### Buat Domain gratis

Tiap pengguna dapat membuat nama domain secara gratis menggunakan subdomain sebagai contoh seperti 
**namaweb.landingin.id**, namaweb adalah nama domain anda sebagai subdomain dan dilanjuti dengan domain utama landingin.id.

Jumlah maksimal dalam membuat subdomain tergantung layanan paket yang dipilih silahkan lihat lagi pada halaman harga.

### Menambahkan Domain

Cara untuk menambahkan domain silahkan pergi ke halaman **landing page -> klik button Tambah Baru**, maka akan muncul halaman
seperti screenshot dibawah ini :

<img src="/uploads/tambah-domain-1.png" alt="Alt text" width="80%" />

Tentukan nama subdomain sebelum masuk ke proses selanjutnya

<img src="/uploads/tambah-domain-2.png" alt="Alt text" width="80%" />

Apabila sudah silahkan klik button "Go Online", maka proses instalasi landing page
otomatis berjalan dan tunggu beberapa menit.

<img src="/uploads/tambah-domain-3.png" alt="Alt text" width="100%" />

Dan apabila sudah selesai anda akan diarahkan ke halaman landing page kembali dan 
landing page sudah online.
