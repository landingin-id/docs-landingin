### Upload Produk

Tiap produk anda dapat mengupload maksimal 5 gambar yang nantinya akan tampil berupa
gambar slideshow pada halaman link produk anda.

### Upload Attachment Produk

Fitur untuk produk digital agar pelanggan dapat mendownload resource produk anda, apabila
telah melakukan pemesanan dan pembayaran. opsi upload attachment terdiri dari :

* <b>Lokal Komputer :</b> apabila anda ingin mengupload langsung file attachment ( contoh pdf )
langsung dari lokal komputer anda.
* <b>Gdrive :</b> apabila anda ingin telah mengupload file attachment ( contoh pdf )
melalui platform gdrive.
* <b>Dropbox :</b> apabila anda ingin telah mengupload file attachment ( contoh pdf )
melalui platform dropbox.
* <b>Others :</b> apabila anda ingin telah mengupload file attachment ( contoh pdf )
melalui platform yang lain.
