### Penjelasan

Setiap anda menambahkan produk baru, link produk akan otomatis dibuat dan dapat anda gunakan
untuk di share di sosial media atau untuk kebutuhan pemasangan iklan penjualan, sebagai
contoh

#### Memasang Produk Link ke landing Page

Dari halaman landing page. agar pelanggan dapat diarahkan ke halaman link produk anda dan langsung dapat melakukan checkout produk.
