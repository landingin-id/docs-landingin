### Penjelasan

Apabila pelanggan anda merasa gaptek, anda dapat membuatkan order terkait produk anda untuk pelanggan anda yang nantinya akan menggenerate sebuah payment link dan dapat anda share
ke nomor whatsapp pelanggan untuk kebutuhan followup dll. tahapan membuatkan order yaitu :    .


### Ke halaman produk

pada halaman produk klik button "Open Link Produk" maka akan muncul popup window browser baru

#### Klik Pemesanan

Pada halaman popup klik button "Pesan Sekarang"

#### Tentukan Metode Pembayaran

Dihalaman selanjutnya tentukan metode pembayaran lalu klik button "Bayar Sekarang"

<img src="/uploads/checkout-pilihan-pembayaran.png" alt="Alt text" width="100%" />

#### Link Pembayaran

Selanjutnya link pembayaran berhasil dibuat, dan anda dapat menshare url link pembayaran ke pelanggan anda.
