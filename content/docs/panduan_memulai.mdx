### 1. Tentukan Layanan

Landingin terdiri dari 3 layanan untuk pengguna, yang diantaranya terdiri dari Layanan Node Basic, Node Creator dan Node Pro.
dari masing masing layanan tersebut terdapat opsi diantaranya :

  <Cards>
    <Card href="/docs/Layanan#opsi-layanan" title="Uji Coba 7 Hari">
      Pilihan uji coba fitur selama 7 hari untuk jenis layanan berbayar.
    </Card>

    <Card href="/docs/Layanan#opsi-layanan" title="Langsung Berlangganan">
      Anda akan diarahkan ke halaman pembayaran terlebih dahulu, setelah proses menentukan domain
    </Card>
  </Cards>

### 2. Tentukan Domain

Tentukan nama domain sebelum lanjut ke tahapan berikutnya, nama domain berupa subdomain contoh
example.landingin.id, yang nantinya dapat kamu [custom nama domain](/docs/fitur_kustom_domain). dengan nama domain yang telah anda tentukan.


### 3. Setup Landing Page

Proses setup berlangsung secara otomatis, dimulai dari pendaftaran nama domain, konfigurasi database, setup https, konfigurasi editor dan sampai
Landing Page anda siap publish dan dapat diakses. Tunggu proses ini memerlukan waktu beberapa menit.


### 4. Go Online

Landing page telah siap digunakan, anda dapat merancang ulang konten serta desain awal yang sudah dibuat secara otomatis.
Menggunakan [editor landingin](/docs/fitur_editor) yang telah tersedia.
