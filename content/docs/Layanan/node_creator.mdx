### Tentang produk

Node Creator adalah layanan paket berbayar yang kami hadirkan untuk para pelaku bisnis digital baik marketer / advertiser yang sudah mulai menghasilkan cuan. 
<b>Uji coba 7 hari gratis</b> bisa anda coba tanpa komitmen sebelum memutuskan untuk berlangganan.

### Fitur diberikan :

1. Fitur di Node Basic
2. Kustom Domain
3. Template Galeri
5. Email Notification
6. Statistik Pengunjung


### Limitasi Fitur :

Pada Node Creator perbedaan ada pada jumlah limitasi yang diberikan, anda bisa melihat detailnya dibawah ini :


| Fitur             | Limitasi       |
| ------------------- | ------------ |
| Domain              | 2            |
| Toko                | 2 Toko           |
| Halaman      | 8 Landing Page           |
| Produk              | 8 Produk Upload           |
| Order              | 1000 Order perbulan          |
| Kustom Domain              | Tersedia          |

### Penjelasan Limitasi

1. <b>Domain</b> : anda hanya memiliki 2 subdomain / domain pada layanan ini.
2. <b>Toko</b> : Setiap domain akan memiliki toko secara otomatis yang dapat di personalisasikan.
3. <b>Halaman</b> : Kami memberikan limitasi jumlah landing page yang bisa dibuat pada layanan ini.
4. <b>Produk</b> : Jumlah produk yang dapat di upload di dashboard untuk di tampilkan di etalase toko.
5. <b>Domain</b> : Kami memberikan limitasi jumlah orderan yang dapat diterima dalam setiap bulanya.
6. <b>Kustom domain</b> : Ganti nama domain dengan extension seperti .com, .id, .net dan lain lain.
