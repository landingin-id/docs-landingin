### Tentang produk

Node Basic adalah layanan paket gratis yang kami hadirkan untuk anda yang baru saja memulai dan ingin mencoba layanan kami.
tujuan dihadirkanya layanan ini ialah untuk membantu para pelaku UMKM dapat memiliki halaman
personal bisnisnya secara mandiri tanpa perlu mengeluarkan biaya sepeserpun. Dan tentu dengan tampilan
antarmuka halaman yang proper dan mudah di kustomisasi pastinya.

### Fitur diberikan :

1. Editor Landing Page
2. Kustomisasi Toko
3. Checkout Page
4. Fitur Instant Payment
5. Fitur Pencairan
6. Payment Link


### Limitasi Fitur :

Layanan ini tentu memiliki beberapa limitasi yang kami terapkan di dalam dashboad.
anda bisa lihat detailnya dibawah ini :


| Fitur             | Limitasi       |
| ------------------- | ------------ |
| Domain              | 1            |
| Toko                | 1 Toko           |
| Halaman      | 2 Landing Page           |
| Produk              | 4 Produk Upload           |
| Order              | 20 Order perbulan          |

### Penjelasan Limitasi

1. <b>Domain</b> : anda hanya memiliki 1 subdomain pada layanan ini. tidak bisa melakukan
penambahan.
2. <b>Toko</b> : Setiap domain akan memiliki toko secara otomatis yang dapat di personalisasikan.
3. <b>Halaman</b> : Kami memberikan limitasi jumlah Landing Page yang bisa dibuat pada layanan ini.
4. <b>Produk</b> : Jumlah produk yang dapat di upload di dashboard untuk di tampilkan di etalase toko.
5. <b>Domain</b> : Kami memberikan limitasi jumlah orderan yang dapat diterima dalam setiap bulanya.
