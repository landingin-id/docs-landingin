### Tentang produk

Node Pro adalah layanan paket berbayar yang kami hadirkan untuk para pelaku bisnis digital baik marketer / advertiser yang sudah mahir dan berpengalaman. 
<b>Uji coba 7 hari gratis</b> bisa anda coba tanpa komitmen sebelum memutuskan untuk berlangganan.

### Fitur diberikan :
1. Fitur di Node creator
2. Akses template premium
3. Instal banyak template


### Limitasi Fitur :

Pada Node Creator perbedaan ada pada jumlah limitasi yang diberikan, anda bisa melihat detailnya dibawah ini :


| Fitur             | Limitasi       |
| ------------------- | ------------ |
| Domain              | 6            |
| Toko                | 6 Toko           |
| Halaman      | Unlimited           |
| Produk              | 48 Produk Upload           |
| Order              | 10000 Order perbulan          |

### Penjelasan Limitasi

1. <b>Domain</b> : anda hanya memiliki 6 subdomain / domain pada layanan ini.
2. <b>Toko</b> : Setiap domain akan memiliki toko secara otomatis yang dapat di personalisasikan.
3. <b>Halaman</b> : Kami tidak memberikan batasan jumlah halaman yang dapat dibuat pada layanan ini.
4. <b>Produk</b> : Jumlah produk yang dapat di upload di dashboard untuk di tampilkan di etalase toko.
5. <b>Domain</b> : Kami memberikan limitasi jumlah orderan yang dapat diterima dalam setiap bulanya.
